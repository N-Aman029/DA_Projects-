# DA_Projects-

1. Project Title
⭐Balaji Store Sales Dashboard :
This is Power BI Sales Dashboard Project.

2. Short Description :
This Power BI Sales Dashboard analyzes store performance using interactive visuals for sales, profit, quantity, and delivery metrics. It tracks monthly trends, regional distribution, payment modes, shipping types, and product categories to identify top-performing segments and business patterns. The dashboard helps stakeholders quickly monitor KPIs, compare performance across states and years, and support data-driven decision-making for improving revenue and operational efficiency.

3. Tech Stack :
∎ Power BI Desktop – data modeling, DAX measures, interactive visualizations
∎ Power Query (M Language) – data cleaning, transformation, shaping
∎ DAX (Data Analysis Expressions) – KPIs, calculated columns & measures
∎ CSV Dataset – data source (Chatgpt)

4. Data Source :
∎ Synthetic dataset generated using ChatGPT – realistic retail sales data      created for learning and portfolio purposes
∎ Format: CSV
∎ Contains fields like: Order Date, Ship Mode, Region, State, Category, Sub-Category, Sales, Quantity, Profit, Payment Mode, Segment
∎ Used only for demonstration & analysis (not real company data)

5. Features / Highlights :
∎ Interactive KPI cards: Sales, Profit, Quantity, Avg Delivery Time
∎ Monthly Sales & Profit trend analysis across multiple years
∎ Regional performance comparison (North, South, East, West, Central)
∎ Category & Sub-category performance insights (Technology, Furniture,         Office Supplies)
∎ Customer segment distribution (Consumer, Corporate, Home Office)
∎ Payment mode analysis (UPI, COD, Cards, Net Banking, Online)
∎ Shipping mode efficiency comparison
∎ State-wise performance visualization
∎ Fully dynamic filters & slicers for drill-down analysis
∎ Designed for quick business decision-making and performance monitoring
